# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in OpenLaunchKit, please do **not** open a public GitHub issue.

Instead, email: **security@openlaunchkit.dev** (or open a GitHub Security Advisory).

**Please include:**
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and aim to release a patch within 7 days for confirmed vulnerabilities.

## Scope

This tool reads local files but does not transmit any data. It has no runtime dependencies and requires no network access.
