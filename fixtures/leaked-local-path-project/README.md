# data-pipeline

A data processing pipeline tool.

## Installation

```bash
npm install data-pipeline
```

## Usage

Run the pipeline from your project directory:

```bash
node /Users/john/projects/data-pipeline/bin/run.js --config config.json
```

For Windows users, the config is located at:
`C:\Users\dev\AppData\Local\data-pipeline\config.json`

The tool expects data files in `/home/ubuntu/datasets/` by default.

## Configuration

Edit the config at `/Users/john/projects/data-pipeline/config.json` to set your preferences.
